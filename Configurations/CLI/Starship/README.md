

Add this script into `~/.zshrc` or `~/.bashrc` depending if you are using **zsh** or **bash** config file.

```bash
############################## STARSHIP ##############################
# starship file conf here : ~/.config/starship.toml
eval "$(starship init zsh)"

############################## STARSHIP ##############################
```

> apparently this behavior is same for Unix (i just have tested on Unix OS (Debian, MacOS))
